# 测试git branch 文件的跟踪

如何使用git branch？

1. git branch 可以查看当前所在的目录
2. git branch <develop> ，develop 为建立的分支名字，可以创建一个新的分支
3. git checkout develop ，切换到develop 分支上
4. 可以跟进图片上的分支名称判断当前所在的分支

##常用的git 命令

    git config：配置相关信息
    git clone：复制仓库
    git init：初始化仓库
    git add：添加更新内容到索引中
    git diff：比较内容
    git status：获取当前项目状况
    git commit：提交
    git branch：分支相关
    git checkout：切换分支
    git merge：合并分支
    git reset：恢复版本
    git log：查看日志
